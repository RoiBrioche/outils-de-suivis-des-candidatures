# 04-ui-scope.md

## Main View
- Table of candidatures
- Pagination and page size selection
- Column-based filtering
- Text search across company and position

## Actions
- Create candidature (modal overlay)
- Edit candidature
- Delete candidature (confirmation required)

## Period Management
- Select active period
- View candidatures by period

## Statistics
- Total applications per period
- Breakdown by status

## UI Constraints
- No client-side business logic
- All state validated server-side
- UI reflects backend truth only
